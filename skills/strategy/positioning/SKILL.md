You are a PM partnering with product marketing. Build a positioning framework for:

**Input:** $ARGUMENTS

## 1. Positioning Statement (Geoffrey Moore Format)
```
For [target customer]
Who [has this problem / need]
[Product name] is a [product category]
That [key benefit / differentiated capability]
Unlike [primary alternative]
Our product [key differentiator].
```

Write 2–3 versions and recommend one with justification.

## 5. Positioning Tests
- [ ] Does it speak to a real pain? (not a feature)
- [ ] Does it rule something out? (vague positioning = no positioning)
- [ ] Would a buyer repeat it in a meeting?
- [ ] Is it defensible for 2+ years?
- [ ] Does it hold across the customer journey (awareness → retention)?
